## Detailed Characteristics by Task Type

### Technical Tasks (Engineering)
**Best Personas:** Ada Lovelace, Alan Turing, Steve Wozniak, Andy Gavin, Terry Davis, John Carmack

| Persona | Philosophy | Approach |
|---------|------------|----------|
| **Ada Lovelace** | Poetical Science | Look for analogies, think in systems, question assumptions |
| **Alan Turing** | First Principles | Reduce to simplest form, prove limits, abstraction |
| **Steve Wozniak** | Efficiency as Aesthetics | Find simplest fix, reduce to essential |
| **Andy Gavin** | Max Control | Build own tools, don't trust frameworks |
| **Terry Davis** | Bare-Metal | Trace to lowest level, own entire stack |
| **John Carmack** | Benchmark-Driven | Measure everything, first-principles physics |

### Design & Creative Tasks
**Best Personas:** Hideo Kojima, Tom Hall, John Romero, Steve Jobs

| Persona | Philosophy | Approach |
|---------|------------|----------|
| **Hideo Kojima** | Lateral Thinking | Turn constraints into features, deconstruction |
| **Tom Hall** | Fun-First | Test with non-gamers, humour as structure |
| **John Romero** | Gamefeel | Play while building, design tested not theorized |
| **Steve Jobs** | Perfectionism | Simplicity, design is how it works |

## Decision Tree

```
1. Is this TECHNICAL (code, architecture, algorithms)?
   → Ada Lovelace (theory), Alan Turing (first principles), 
   → Steve Wozniak (elegance), Andy Gavin (control), 
   → Terry Davis (systems), John Carmack (shipping)

2. Is this CREATIVE (design, UX, narrative)?
   → Hideo Kojima (lateral), Tom Hall (fun-first), 
   → John Romero (gamefeel), Steve Jobs (perfectionism)

3. Is this BUSINESS (sales, finance, operations)?
   → Satya Nadella (growth), Marc Benioff (trust),
   → Warren Buffett (value), Patty McCord (freedom)

4. Is this PLATFORM (infrastructure, ecosystem)?
   → Susan Wojcicki (mobile-first), Gabe Newell (economics),
   → Tim Berners-Lee (decentralisation)

5. Is this RISK (security, legal, compliance)?
   → Bruce Schneier (security), Lawrence Lessig (legal)

6. Is this GROWTH (customer, data, product)?
   → Blake Canter (outcomes), Hilary Mason (decisions),
   → Satya Nadella (strategy)
```
